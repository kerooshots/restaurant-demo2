# Alforno's Italian Kitchen — Italian Heritage Concept

A responsive one-page concept redesign built with HTML, CSS, and JavaScript.

## Design direction
- Modern interpretation of Italian heritage
- Historical typography, arches, seals, Roman numerals, and subtle tricolore details
- No stock restaurant-interior photography
- Food imagery is limited to imagery already used on Alforno's current official website

## Publish with GitHub Pages
1. Upload all files to the root of the repository.
2. Open Settings → Pages.
3. Select Deploy from a branch.
4. Choose main and /(root), then save.
